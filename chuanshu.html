使用手册

🌌 什么是穿书系统

穿书系统是星叙的配套工具，让你以主角或旁观者身份真正进入小说世界，和书中人物互动，甚至改变故事的走向。你输入的每一句话都是你在故事里的行动，AI会以小说作者的笔法接续剧情，打造出完全属于你的专属故事。

🚀 快速上手

第一步：选书 — 点击底部 + 按钮，支持三种来源：

  • 📚 已完结收藏：星叙收藏夹里的完结长篇（1万～20万字）
  • ✍️ 我的创作：你在星叙里写的小说
  • 📄 导入TXT：从手机本地导入文本文件

同一本书可以创建多个旅途，尝试不同的剧情分支。创建后在卡片菜单里点「重命名」起个专属名字（如"拯救女主线"），方便区分。

第二步：设定身份 — 进入「设置 → 用户身份」，描述你穿越后的角色（名字、性别、与主角的关系等）。这是全局设定，对所有旅途生效。身份可以保存为预设，方便在不同旅途间切换。

第三步：开始旅途 — 选好书后点「开始旅途」，AI会先描述故事开场。然后你在输入框里发送行动，比如"我走向书桌翻开那封信"，故事就此展开。

💬 对话技巧

  • 行动描述：直接写你在做什么，比如"我转身离开，没有回头"
  • 说出台词：用引号包裹，比如 "你说谎了"，AI会自动接上对方反应
  • 心理活动：写内心想法，AI会以旁白体描述你的情绪
  • 强推剧情：想快进剧情可以直接说"跳到三个月后"或"总结一下这段时间发生的事"
  • 输入防丢：你在输入框写的内容会自动保存为草稿，切换页面或刷新后恢复，发送成功后自动清除

⌨️ 快捷键与快捷短语

对话框上方的胶囊栏提供三类快捷操作：

  • ✂️ 分隔按钮：黄色胶囊，点击后输入文字（如"三天后""场景切换"），在对话中插入一条时间/场景分割线，帮你标记剧情节点
  • 快捷键：紫色胶囊，点击快速插入括号、引号等常用符号。在「设置 → 快捷键设置」中自定义。编辑时可插入 {光标} 标记控制光标位置，长按胶囊可拖动调整排列顺序
  • 快捷短语：绿色胶囊，点击直接发送预设指令（如"继续""详细描写"）。在「设置 → 快捷短语」中自定义名称和发送内容

🔄 AI回复多版本

对AI回复不满意？不需要丢弃重来：

  • 长按AI消息，选择「重新生成」，AI会生成新版本
  • 原版本自动保留，消息上方出现版本切换栏（如"版本 1/2"）
  • 点左右箭头在不同版本间切换，找到最满意的走向
  • 注意：重新生成后，该消息之后的对话会被移除（因为剧情已分叉）

🧠 记忆管理

AI每次回复只能记住最近几十轮对话，时间一长会"遗忘"前期剧情。点顶部栏的大脑图标打开记忆管理：

  • 故事摘要：点「AI生成摘要」自动总结进度，也可手动编辑。摘要始终发送给AI，确保长期记忆
  • 自动总结：每隔 N 轮AI回复会自动在后台更新摘要，间隔可调节（默认与上下文轮数一致）
  • 关键事实锁定：写下绝对不能遗忘的事实（如"男主已知女主身份"），始终强制发送给AI
  • 上下文轮数：控制每次发送给AI的最近对话条数，可按旅途单独调节
  • 对话显示条数：控制页面上显示的消息数量，对话量大时建议设为50~100避免卡顿
  • 全书梗概：让AI生成全书梗概，启用后用梗概替代原文发送，大幅节省Token（注意检查梗概完整性）

💾 存档系统

在旅途选项或长按消息菜单中操作：

  • 创建存档：在当前位置或指定消息处保存快照，最多20个
  • 加载存档：回滚到任意存档点，之后的对话被替换。建议先保存当前进度再回滚
  • 配合多版本生成和多旅途功能，可以充分探索不同剧情分支

🔖 书签与时间线

  • 点击AI消息右上角的书签图标，为关键情节做标记（如"初遇女主""决战前夜"）
  • 点顶部栏的时间线图标，打开侧边抽屉，纵览所有书签和存档点
  • 点击节点可快速跳转到对应位置，方便回顾剧情脉络

🎭 写作预设（人设 / 世界观 / 文风）

在「设置 → 写作预设管理」里创建三类预设：

  • 人设：主要人物的性格、外貌、技能、背景等
  • 世界观：时代背景、修炼体系、特殊规则等
  • 文风：写作风格要求，如"文笔细腻，多写心理描写，节奏慢热"

预设创建后，可以在编辑页面直接快速绑定到某本书或某个系列。也可以在「设置 → 设定绑定」中统一管理。每本书/系列每类只能绑定一个预设，但同一个预设可以绑定多本书。

🔗 系列关联

如果一套故事有多部（比如三部曲），可以关联成一个系列：

  • 在旅途卡片菜单里点「关联系列」
  • 按顺序添加各部小说，拖动调整顺序，输入系列名称
  • 双击书架上的系列名称可快速修改
  • 系列支持自定义文件夹颜色和整体置顶
  • 系列共享同一套绑定设定，前作总结可自动传递给续作

📖 阅读与导出

在旅途选项中可以找到：

  • 小说阅读模式：全屏沉浸阅读，隐藏用户输入，自动分章节。支持字号、行距调节和横排/竖排切换
  • 导出排版小说：将AI叙事部分导出为精美排版的HTML网页或Markdown文件，自动分章
  • 导出旅途记录：导出为TXT纯文本，包含所有对话
  • 搜索对话：按关键词搜索历史对话，快速定位到目标消息

⏸️ 停止与恢复旅途

  • 在旅途选项中可以停止旅途，停止后无法继续对话，记录保留
  • 如果误操作，停止后底部会出现「恢复旅途」按钮，点击即可重新激活
  • 也可以清除所有对话重新开始，设定绑定不受影响

🎨 页面美化

在「设置 → 页面美化」中自定义外观：

  • 主题颜色：深紫、暖橙、翠绿、冰蓝、浅色五种主题
  • 对话字体：四种内置字体 + 支持自定义Google Fonts链接
  • 字距/词距：滑块微调文字间距
  • 对话背景图：选择图片作为对话区域背景
  • 全屏沉浸模式：在设置页点击进入/退出全屏，隐藏地址栏和状态栏

🗃️ 数据备份

穿书系统有独立的数据备份功能：

  • 在「设置 → 导出备份」中，将所有旅途、设定、预设、快捷键、快捷短语等导出为一个JSON文件
  • 在「设置 → 导入备份」中，选择JSON文件恢复数据（会覆盖当前数据）
  • 换手机、清缓存、或想在不同设备间同步时使用
  • 建议定期备份，避免数据丢失

穿书系统的所有数据（旅途、设定、预设等）也包含在星叙的完整备份中，在星叙里导出备份时会自动携带穿书数据，导入时也会一并恢复。

📊 旅途数据统计

在旅途选项的顶部可以看到统计面板：

  • AI叙事字数、你的输入字数、对话轮数的详细统计
  • 可视化的AI/用户占比柱状图

💡 玩法建议

  • 蝴蝶效应流：选一本你熟悉的书，在关键节点做出和原著不同的选择，看故事会往哪里走。用存档系统在关键抉择前保存，探索多个分支
  • 多线并行流：对同一本书创建多个旅途并重命名（如"善人线""恶人线"），同时推进不同路线
  • 自定义主角流：在用户身份里设定一个全新的穿越者身份，以旁观者视角影响原著主线
  • 复盘阅读流：精读一本书后穿越，检验自己对人物和世界观的理解
  • 原创创作流：用你自己在星叙写的小说穿越，AI会深度配合你的原创设定
  • 版本对比流：在AI的关键回复上反复重新生成，切换不同版本对比，挑选最佳走向

🎚️ 对话节奏控制

不同场景需要不同的叙事节奏。在旅途选项中可以调节AI的回复长度：

  • 简短（80~150字）：适合快节奏对话、日常闲聊、紧张追逐等场景
  • 标准（180~350字）：默认模式，适合大多数剧情推进
  • 详细（350~600字）：适合大场景描写、关键转折、情感高潮等需要铺陈的段落

每本书的回复长度独立设置，互不影响。也可以打开「全局通用」开关，让所有旅途共享同一个长度设定。随时可以切换，比如战斗场景用简短，风景描写用详细。

📋 剧情回顾生成

隔了几天再回来继续故事，忘了上次玩到哪了？

  • 在旅途选项中点击「生成剧情回顾」（需至少6条对话）
  • AI会用3~5句话概括你当前的故事进度，语气像朋友在提醒你"上回书说到哪了"
  • 回顾生成后会显示在对话区顶部，打开旅途时自动可见
  • 点击回顾卡片右上角的 × 可以关闭
  • 每次点击「生成剧情回顾」都会重新生成最新的回顾内容

🌳 对话分支树

穿书系统支持通过存档探索不同剧情分支。分支树帮你直观了解"我在哪条线上"：

  • 点击顶部的时间线图标，在侧边抽屉中切换到「分支树」标签页
  • 每个存档点显示为一个节点，按对话位置从上到下排列
  • 当前所在位置用高亮 + ◀ 标记标注
  • 点击任意存档节点可以加载该分支，回到那个剧情岔路口
  • 玩法建议：在关键抉择前创建存档，然后大胆做选择。想试另一条路时，回到分支树点击之前的存档即可

🎲 行动建议（灵感骰子）

不知道该做什么？让AI给你出主意：

  • 在对话输入框左侧点击 + 按钮，选择「灵感骰子」
  • AI会根据当前剧情，给出3条行动建议，覆盖行动、对话、内心观察三个方向
  • 点击任意一条建议，内容自动填入输入框，可以编辑后再发送
  • 点击「换一批」可以重新摇骰子，获取新的灵感
  • 灵感建议不会直接发送，你始终有最终决定权

🌍 世界状态面板

故事推进过程中，AI会自动追踪并记录当前的世界状态：

  • 在对话输入框左侧点击 + 按钮，选择「世界状态」即可查看
  • 当前时间：故事进行到什么时间点
  • 当前事件：正在发生的关键剧情
  • 人物状态：各角色当前的位置、与你的关系和态度
  • 关键物品：重要道具的持有情况
  • 状态信息在每次AI回复后自动更新，无需手动维护
  • 适合在做决策前快速确认"现在谁在哪、发生了什么"

⚙️ API 设置说明

  • 穿书系统默认读取星叙的API配置，也可以在「设置 → API设置」中单独配置
  • 支持选择星叙已有的API预设，一键填入
  • 支持拉取模型列表：填好地址和Key后点击「拉取」按钮，自动获取可用模型
  • 测试连接：保存前可点击测试，确认配置正确
  • 创意度推荐 0.7~0.9，过高可能导致故事跑偏，过低则缺乏新意

🔀 AI回复多版本管理

对AI回复不满意？可以反复重新生成，保留所有版本：

  • 长按AI消息，选择「重新生成」，AI会生成新版本
  • 原版本自动保留，消息上方出现版本切换栏（如"版本 1/3"）
  • 点左右箭头在不同版本间切换，对比哪个版本走向最好
  • 点版本栏右侧的🗑️ 按钮可以删除当前版本，其他版本保留不变
  • 注意：重新生成后，该消息之后的对话会被移除（因为剧情已分叉）

📌 分隔标记

对话框上方有一个黄色的「✂️ 分隔」按钮：

  • 点击后输入文字（如"三天后""场景切换到皇宫"），在对话中插入一条分割线
  • 分割线用来标记时间跳转或场景变化，让对话历史更清晰
  • 点击分隔线可以删除它
  • 分隔线不会发送给AI，不消耗Token

📱 消息管理技巧

  • 折叠/展开：长按消息选择「折叠」，可以收起不重要的长段落，点击「展开」恢复
  • 批量删除：长按消息选择「批量选择」，勾选多条后一键删除
  • 改写：长按消息选择「改写」，直接在原位编辑文字内容
  • 输入草稿自动保存：输入框中正在编写的内容会自动保存，切换页面或刷新后自动恢复，发送成功后自动清除
  • 加载更多：对话量大时页面只显示最近若干条，向上滑动或点击「加载更早消息」可查看历史

📖 小说阅读模式详解

在旅途选项中点击「小说阅读模式」进入沉浸式阅读：

  • 自动将AI叙事内容按每3000字分为一章，自动编号
  • 隐藏所有用户输入，呈现纯小说阅读体验
  • 支持字号、行距调节（点击右上角滑块按钮）
  • 支持横排/竖排切换
  • 点击右上角目录按钮可展开章节目录，点击章节快速跳转
  • 自动使用「页面美化」中设置的自定义字体
  • 可单独设置阅读模式背景图（在「页面美化」中设置），与对话背景图互不影响
  • 点击屏幕中央区域可隐藏/显示顶部工具栏

🎨 页面美化详解

在「设置 → 页面美化」中可以自定义：

  • 主题颜色：深紫、暖橙、翠绿、冰蓝、浅色五种主题
  • 对话字体：四种内置字体 + 支持自定义Google Fonts链接，也可一键同步星叙正在使用的字体
  • 对话字号：滑块调节对话区域文字大小（12~22px）
  • 字距/词距：滑块微调文字间距，适应不同字体
  • 对话背景图：选择图片作为对话区域背景
  • 阅读模式背景图：单独设置小说阅读模式的背景，与对话背景互不干扰
  • 全屏沉浸模式：在设置页点击进入/退出全屏，隐藏地址栏和状态栏
  • 所有美化设置可点击「恢复默认」一键重置

🌍 世界状态自动追踪

AI每次回复后会自动提取并更新当前世界状态：

  • 在对话输入框左侧点击 + 按钮 → 世界状态 查看
  • 自动追踪：当前时间、正在发生的事件、人物位置和态度、关键物品持有情况
  • 状态标签对用户不可见，不会显示在对话中
  • 状态标签也不会占用上下文Token
  • 适合在做关键决策前快速确认"现在是什么情况"

🎲 灵感骰子详解

不知道下一步该怎么做？让AI给你灵感：

  • 点击输入框左侧 + 按钮 → 灵感骰子
  • 点击「给我灵感」按钮，AI根据当前剧情生成3条建议
  • 3条建议分别覆盖行动、对话、观察三个方向
  • 点击建议后内容自动填入输入框，可以编辑后再发送
  • 点击「换一批」重新摇骰子
  • 灵感只是建议，你随时可以无视它走自己的路

🔗 关联系列详解

适用于三部曲、系列小说等多部关联作品：

  • 在旅途卡片菜单点「关联系列」，添加各部小说并拖动排序
  • 输入系列名称，所有成员自动归入同一文件夹
  • 双击书架上的系列名称可快速修改
  • 支持自定义文件夹颜色（6种可选）和系列整体置顶
  • 系列共享同一套写作预设绑定（人设/世界观/文风）
  • 前作的故事摘要可自动传递给续作，保持世界观延续

📊 旅途数据统计

在旅途选项的顶部可以看到详细统计面板：

  • 总字数：AI叙事 + 你的输入
  • 分项统计：AI叙事字数、你的输入字数、对话轮数
  • 可视化占比：AI/用户字数的柱状图和百分比

🎚️ 回复长度控制详解

在旅途选项中调节AI每次回复的长度：

  • 简短（80~150字）：快节奏对话、日常闲聊、紧张追逐
  • 标准（180~350字）：大多数剧情推进（默认）
  • 详细（350~600字）：大场景、关键转折、情感高潮
  • 每本书独立设置，也可打开「全局通用」开关让所有旅途共享同一设定
  • 开关切换时会自动同步当前值，避免长度突变

⏸️ 停止/恢复/清除旅途

  • 停止旅途：在旅途选项中操作，停止后无法继续对话，记录保留
  • 恢复旅途：停止后底部出现「恢复旅途」按钮，点击即可重新激活
  • 清除所有对话：删除全部对话记录、摘要、存档和书签，但保留设定绑定，重新开始故事
  • 删除旅途：在书架卡片菜单操作，完全删除旅途，相关绑定自动解除

🖼️ 封面管理

  • 在书架卡片菜单点「更换封面」可自定义旅途封面
  • 支持从手机相册选择图片，自动压缩
  • 从收藏夹选书时会自动带入原书封面

🔧 系统提示词专家模式

高级用户可以直接查看和编辑发送给AI的完整系统提示词：

  • 在旅途选项中点击「系统提示词（专家模式）」
  • 点击「生成当前提示词」查看系统默认生成的完整内容
  • 开启自定义开关后，系统将完全使用你编辑的提示词
  • 原著信息、记忆摘要、上下文对话仍会正常附带
  • 回复质量变差时可点击「恢复默认」回到系统状态

📑 章节标记与场景快照

对话框上方的黄色「✂️ 分隔」按钮已升级为章节标记系统：

  • 点击后输入章节/场景名称（如"第一章：初入书院"）
  • 系统自动记录当时的世界状态快照（时间、人物、事件）
  • 点击对话中的章节标记，可查看当时的世界状态快照
  • 有章节标记时，阅读模式会按标记分章，形成真正的章节目录
  • 无章节标记时仍按字数自动分章

✍️ AI重写段落（对比展示）

对AI的某段叙事不满意但不想丢弃：

  • 长按AI消息，选择「重写本段」
  • AI会在对比面板中重写同一情节的不同版本
  • 原文和重写版本并排展示，点击选中满意的版本
  • 点击「再写一版」可以继续生成更多风格（最多5版）
  • 选中后点击「使用选中版本」，自动替换并保存到版本系统
  • 区别于「重新生成」：重写不会删除后续对话，只替换当前这一条

⚔️ 卡牌对决系统

与书中角色来一场策略对决，AI会根据角色性格智能出牌：

  • 在对话输入框左侧点 + 按钮 → 卡牌对决，或AI叙事中出现战斗场景时自动提示
  • 从世界状态中的角色列表选择对手，角色的好感度会影响出牌策略
  • 每回合从手牌中选一张出牌，AI根据角色性格、好感度和局势智能应对
  • 攻击牌：横斩、穿刺（无视闪避）、烈焰术（附带灼烧）、寒冰箭、雷击
  • 防御牌：铁壁（减伤）、闪避（完全回避）、反击（防御+反弹伤害）
  • 辅助牌：回春（恢复HP）、蓄力（下回合攻击翻倍）、淬毒（持续掉血）、洞察（看破对方下一手）
  • 击倒对方（HP归零）即获胜，无回合数限制
  • 随时可以点「认输退出」，HP较多的一方判定获胜
  • 对决结束后可选择「写入故事」，AI会用小说叙事的方式描写整场对决
  • 对决结果会影响角色好感度：胜负关系会根据对方好感度产生不同程度的变化
  • 好感度≥60的角色可能会故意示弱、选温和的牌
  • 好感度＜20的角色会全力以赴、毫不留情

❤️ 角色好感度养成

世界状态中的角色拥有好感度系统，你的行为会影响好感度变化：

  • 好感度范围 0~100，AI每次回复后自动追踪并更新
  • 在世界状态面板中可以看到每个角色的好感度进度条和标签
  • 好感度标签：亲密（≥80）、友好（≥60）、普通（≥40）、冷淡（≥20）、敌对（＜20）
  • 当好感度达到 20、40、60、80 四个关键阈值时，会触发好感度事件卡
  • 事件卡会展示一段角色专属的感情故事片段
  • 每个事件提供 3个选择，不同选择会带来不同的好感度变化
  • 选择后会自动作为行动插入对话，AI会根据你的选择继续叙事
  • 每个角色每个阈值只触发一次，不会重复
  • 好感度变化会反过来影响角色在故事中的态度和反应

🎲 命运转盘（随机事件）

每次打开旅途时，有一定概率触发随机命运事件：

  • 需要至少 5 条对话后才会触发
  • 每个旅途每天最多触发一次
  • 事件类型：幸运（好事）、危机（挑战）、神秘（悬念）、日常（生活）
  • 事件卡会描述一个突发场景，例如：捡到神秘信件、遭遇劫匪、偶遇算命先生等
  • 你可以选择「接受命运」将事件融入故事，AI会根据事件内容继续叙事
  • 也可以选择「暂时忽略」跳过这次事件
  • 最近 5 次触发过的事件不会重复出现
  • 接受事件后，对应的行动会自动插入对话，由AI展开后续情节

⌨️ 电脑端操作

  • 按 Esc 键快速关闭任何弹窗、面板、阅读器
  • 输入框中按 Ctrl+Enter（或 Cmd+Enter）直接发送消息
  • 鼠标悬停在卡片、按钮上有高亮交互效果
  • 右键点击消息可打开上下文菜单（等同于手机长按）

🏷️ 旅途标签分类

旅途多了之后可以用标签来分类管理：

  • 在书架卡片右上角菜单点「管理标签」
  • 输入标签名，用顿号、逗号或空格分隔，例如"玄幻、言情、修仙"
  • 每个旅途最多 5 个标签，标签显示在卡片的书名下方
  • 书架页顶部会自动出现标签筛选栏，点击标签快速过滤
  • 搜索栏也支持按标签名搜索
  • 标签名称相同的会自动归为一组，方便统一筛选

📂 快捷短语分组

快捷短语多了之后可以用分组来整理：

  • 在「设置 → 快捷短语」中编辑短语时，可以填写「所属分组」字段
  • 例如填写"节奏控制""场景切换""角色互动"等分组名
  • 同组的短语会折叠在一个绿色组名胶囊下
  • 点击组名胶囊，弹出该组所有短语，点击直接发送
  • 不填分组的短语仍然直接显示在胶囊栏
  • 建议将常用的短语不分组直接显示，不常用的归入分组节省空间

📥 EPUB 电子书导入

除了 TXT，现在还支持导入 EPUB 格式的电子书：

  • 点击底部 + 按钮，切换到「导入EPUB」标签页
  • 选择本地 .epub 文件，系统会自动解析提取文本内容
  • EPUB 是电子书最常见的格式，各平台下载的小说通常都是这个格式
  • 系统会自动按章节顺序提取正文，去除格式标签
  • 同样要求字数在 1万~20万字 之间
  • 解析后可修改书名，点击「开始旅途」即可
  • 如果解析失败，可能是加密的 EPUB 或非标准格式，建议转换为 TXT 后导入

🗑️ 分隔线与章节标记删除

对话中的分隔线和章节标记的删除方式：

  • 长按分隔线或章节标记约0.5秒，出现红色「松手删除」提示
  • 松手即可删除，无需二次确认
  • 电脑端可以右键点击分隔线直接删除
  • 误删后无法恢复，但可以随时重新插入新的分隔标记

📱 消息管理补充

长按消息的上下文菜单现在对所有消息类型统一提供：

  • 复制：复制消息文本到剪贴板
  • 改写：在原位编辑消息内容
  • 重写本段：仅对AI消息有效，在对比面板中重写
  • 删除：删除当前这条消息（AI和用户消息都可删除）
  • 书签 / 折叠 / 在此存档 / 批量选择：第二行操作

AI消息底部还有三个快捷按钮：🗑️删除、✏️续写、🔄重新生成，无需长按即可操作。

💬 对话气泡模式

在「设置 → 页面美化」中可以切换对话显示样式：

  • 小说排版（默认）：传统小说排版风格，AI叙事左对齐，用户输入右对齐
  • 常规气泡：聊天软件风格的气泡样式
  • 圆角气泡：更圆润的气泡外观
  • 尖角气泡：棱角分明的气泡样式

开启气泡模式后还可选择气泡颜色：雾白、粉白、紫黑、蓝白、红黑、橙黄、青绿七种配色。气泡模式和颜色搭配主题使用效果更佳。

📈 旅途趋势图

在设置页顶部的信息卡片上点击，可展开/收起近7日活跃趋势图：

  • 以柱状图展示最近7天每天的消息数量
  • 帮助你了解自己的穿书频率和活跃度
  • 签到状态会显示今日是否有过活动

🎭 角色好感度系统

世界状态面板中的人物状态现在包含好感度信息：

  • AI会自动追踪各角色对穿越者的好感度（0~100）
  • 好感度以进度条+文字标签形式展示（亲密/友好/普通/冷淡/敌对）
  • 颜色会根据好感度自动变化：绿色=高好感，黄色=中等，红色=低好感
  • 你的行动和选择会影响好感度变化
  • 好感度信息在每次AI回复后自动更新

✍️ 剧情走向建议（剧情分析）

除了灵感骰子，还可以获取更深度的剧情走向分析：

  • 在灵感面板中点击「剧情分析」按钮
  • AI会综合当前故事摘要、世界状态和最近对话，给出3条走向建议
  • 每条建议包含：具体行动、预期影响、风格标签
  • 三条建议分别对应：稳妥保守、激进冒险、出人意料三种风格
  • 标签包括：好感提升、关系恶化、剧情加速、埋下伏笔、高风险高回报、稳扎稳打
  • 点击建议可填入输入框，编辑后发送

📋 输入草稿自动保存

输入框中正在编写的内容会自动保存：

  • 切换页面、刷新浏览器后，重新打开旅途时自动恢复草稿
  • 发送成功后自动清除草稿
  • 每个旅途独立保存草稿，互不影响
  • 草稿每1.5秒自动保存一次，无需手动操作

💾 存档备注

创建存档时，现在会弹出两次输入框，让你记录当时的剧情状态：

  • 第一次：给存档起名，如"第三章 城楼相遇"
  • 第二次：填写状态备注，如"好感度38，刚刚冲突，颜如玉有些动摇"——这条备注会以 📌 标注显示在存档卡片上
  • 备注是可选的，直接点确认跳过也没问题
  • 加载存档时一眼就能判断该选哪个，不再靠猜

⏱️ 自动定时存档

每隔 10轮 AI 回复，系统会自动静默创建一个存档，无需手动操作：

  • 存档名格式为「自动存档 5月19日 14:30」，卡片右侧有灰色「自动」标签与手动存档区分
  • 最多保留 5个自动存档，超出后自动删除最旧的那个
  • 生成时会短暂显示「✓ 已自动存档」提示
  • 退出当前旅途后计时器自动停止，不会在后台消耗资源
  • 在存档管理弹窗中可以查看和加载所有自动存档

🗑️ 对话回收站

删除消息后，消息不会立刻消失，而是先进入回收站保留 72小时：

  • 删除后 3秒内底部会出现「↩️ 撤销」悬浮提示，点击可以原地恢复（含书签位置）
  • 超过3秒后消息进入回收站，可在之后随时恢复
  • 进入回收站：点「⋯」旅途选项 → 存档管理 → 底部「🗑️ 回收站」按钮
  • 回收站里可以看到每条消息的内容预览和删除时间，点「↩️ 恢复」将消息追加到对话末尾
  • 回收站最多保留 50条消息，72小时后自动清除
  • 也可以在回收站底部点「🗑️ 清空回收站」一次性清除

✋ 「暂停一下」模式

觉得 AI 推进剧情太快、还没想好下一步？用这个功能让 AI 主动踩刹车：

  • 点击对话输入框左侧 + 按钮 → 暂停一下 ✋
  • AI 会在下一段叙述结尾停在一个自然的停顿点（如人物开口之前、门刚推开之时），不再自己继续往下写
  • 这不会中断生成，只是给 AI 加了一条隐藏指令让它收着点
  • 停下来之后，从容输入你的下一步行动，再发送即可继续

🌿 「如果当时」分支

觉得某个剧情走向走错了，但不想毁掉现在这条时间线？用「如果当时」开辟平行支线：

  • 长按任意一条 AI 消息，弹出菜单第一排最右侧选择「如果当时 ⑂」
  • 系统自动截取到那条 AI 消息之前（你上一次行动之前），把书的所有设定、记忆摘要、世界状态原样复制，创建一条全新的分支旅途
  • 跳转到新旅途后，重新输入那次的行动，故事从岔路口重新出发
  • 原始旅途完全不受影响，两条时间线独立存在，可以随时从书架切换
  • 分支旅途的书卡上会显示 「⑂ 分支自《书名》」 紫色小标签，方便辨认
  • 创建时可以自定义名称，例如"颜如玉支线·温柔结局"
  • 建议定期整理不再需要的分支旅途，避免书架列表过长

✨ 发送按钮图标提示

发送按钮会根据输入框状态自动变换图标，帮助你了解当前点击的效果：

  • ✨ 魔法棒图标（输入框为空时）：点击 = 触发 AI 根据你最后一条消息生成回复
  • ✈️ 纸飞机图标（输入框有内容时）：点击 = 仅发送你写的内容，不触发 AI 回复
  • 发送内容后如需 AI 接续，等输入框清空后再点一次魔法棒即可

隐藏数值追踪系统

穿书系统可以在背后追踪若干关键数值，反映你在故事中的成长轨迹。默认提供三项，可自定义至多4项：

  • 在「设置 → 数值系统 → 隐藏数值系统」中点击，进入全屏设置弹窗
  • 弹窗顶部有启用/禁用开关，开启后AI会在每次回复时自动更新数值
  • 每项数值范围固定为 0～100，AI回复后自动更新，系统会确保数值不会超出此范围
  • 最多可设置 4项数值，点击底部「添加新数值」新增；点击数值卡片右上角删除图标移除（至少保留1项）
  • 每项数值均支持完全自定义：可修改名称（如把"黑化值"改成"邪恶指数"）、图标（12种可选）、颜色（6种可选）
  • 上升/下降条件：直接告诉AI"什么行为让这个数值增加/减少"，AI就会据此自动调整，默认规则可作参考
  • 阈值触发效果：可以写"数值超过XX时会发生什么"，AI会把这条规则融入叙事（如"超过70时NPC主动求助"）
  • 作用范围：全局=通用数值；角色=在世界状态面板中每个角色下单独显示一条进度条；自己=只属于穿越者
  • 修改后点击「保存」生效；点击「恢复默认」可一键重置所有自定义
  • 在「世界状态」面板中可查看数值进度条，样式与好感度进度条统一，数值以「当前值/100」形式显示
  • 数值保存在旅途数据中，导出备份时一并携带

背包/道具系统

在旅途中获得的道具可以收入背包，随时查看和使用：

  • 点击对话输入框左侧 + 按钮 → 背包 打开背包面板
  • 手动添加道具：点击底部「手动添加道具」，依次填入道具名称、描述（可选）和图标Emoji（默认为箱子图标），同名道具会自动叠加数量
  • 使用道具（绿色按钮）：道具以行动形式自动插入对话，AI会根据道具效果续写故事；使用后数量减1，归零则自动移除
  • 赠送道具（橙色按钮）：从世界状态角色中选择赠送对象，好感度自动+3，并触发AI回复
  • 丢弃道具（灰色按钮）：从背包中删除该道具，不插入对话
  • AI在故事中安排你获得新物品时，会在文末提示「获得道具：xxx」，你可以手动添加到背包
  • 每个旅途的背包独立保存，互不影响

角色百科

为旅途中遇到的每个角色建立专属档案，随时查阅和管理：

  • 点击对话输入框左侧 + 按钮 → 角色百科 打开
  • 点击「生成/更新百科」，AI会根据近30轮对话自动分析生成角色档案（至少需要4条对话）
  • 角色以竖向卡牌形式显示，封面展示头像（或姓名首字）、好感度进度条
  • 点击任意角色卡片，进入详情页，可查看好感度变化趋势曲线、经典台词、关系图谱等完整信息
  • 标签管理：在详情页「性格标签」区可管理角色标签。点击标签名称可改名；点击标签右侧「×」可删除；点击右上角「+ 添加」可创建自定义标签（每个角色最多8个）。AI生成的标签同样可以修改
  • 在详情页点击「更换头像」，可从手机相册上传专属图片（自动压缩到200×200像素以内）
  • 点击「更新此角色」，AI会重新分析并刷新该角色的档案内容
  • 点击角色卡片右上角的删除按钮可移除该角色的百科档案
  • 角色分类管理：在百科列表顶部可新建角色分类（点击虚线「+」标签），将角色归入不同组（如男主、女配、反派等），方便分类查看
  • 点击分类标签进行筛选；点击标签右侧 × 图标或长按标签（手机端）可删除该分类（角色档案不会被删除，只取消分类标记）；双击标签名可重命名（电脑端）
  • 角色百科数据独立保存在每个旅途中，导出备份时一并携带

🎴 角色百科详情页操作

在角色卡牌上点击进入详情页，可以进行精细化管理：

  • 写入AI开关：左上角的开关控制是否把该角色信息写入AI上下文。开启后AI叙事时会严格遵守此角色设定
  • 写入下一本：仅在系列文件夹中显示。开启后，此角色信息会在打开下一本书时自动导入新书的角色百科
  • 性格标签：点击标签名可改名，点击右侧 × 删除，点击右上角「+ 添加」新建（每个角色最多8个）
  • 经典台词：点击标题旁的笔图标，每行一条，最多5条
  • 角色简介：点击标题旁的笔图标编辑长文简介
  • 更换头像：从手机相册选择，自动压缩至 200×200 像素
  • 分类：点击姓名下方的分类胶囊切换分类
  • 更新此角色：底部按钮，让AI重新分析并刷新当前角色档案

📂 角色百科批量操作

角色多了之后，可以批量管理：

  • 在角色百科页面右上角点击滑块图标打开批量操作面板
  • 全部写入AI上下文：一键开启或关闭所有角色的「写入AI」开关
  • 全部写入下一本：在系列中一键开启所有角色的"传给续作"标记
  • 删除所有人物档案：清空当前旅途的全部角色百科（角色对话不受影响）

🏷️ 角色百科分类管理

  • 在角色百科列表顶部的标签栏，点击虚线「+」按钮新建分类
  • 点击分类标签筛选该分类下的角色
  • 双击分类标签（电脑端）可重命名
  • 长按分类标签（手机端）可删除该分类（角色档案不会被删除，只会取消分类标记）
  • 每个角色可以归入一个分类，方便区分主角团、反派、配角等不同身份
