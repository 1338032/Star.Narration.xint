## 序言

### 星叙是什么？

星叙 是一款**面向网文创作者的智能写作工具**，运行在浏览器网页中，所有数据保存在本地，不依赖云端账号。

**与普通AI工具的区别：**

- 你永远是作者，AI是搭子——大纲和关键决定永远在你手里
- 设定先行——人物、世界观、规则、禁用词可以精细配置
- 可控生成——通过上下文编辑器、章节大纲、情节分支器精准指挥AI
- 长期陪伴——本地存储 + 备份导出，陪你写完一整本书

**五大核心页面：**

| 页面 | 用途 |
|------|------|
| 🔨 作坊 | 一句话生成长篇/短篇，AI写作主入口 |
| 📚 收藏 | 长篇书架 + 短篇收藏，管理所有作品 |
| ✏️ 创作 | 手动写作区 + 高级创作工具 |
| ⚙️ 设置 | API、人设、世界观、文风、规则等所有配置 |
| 🗂️ 资源 | 官方内置资源包 + 导入/导出，与朋友分享设定 |

## 第一章：搭建你的创作工坊

### 1.1 配置API（最关键的一步）

**什么是API？**

API 是连接星叙和AI服务商的桥梁。你需要自己准备API密钥（通常来自 OpenAI、各类国内代理等），星叙只是帮你组织设定、发送请求，并不提供免费的AI算力。

**设置步骤：**

1. 底部导航点击「设置」→ 进入 **API设置**
2. 填写以下三项：
   - **API地址**：接口基地址，例如 `https://api.openai.com/v1`（部分代理已含 `/v1`，不要重复写）
   - **API密钥**：通常以 `sk-` 开头，注意不要有多余空格
   - **模型名称**：必须与服务商支持的名称完全一致，如 `gpt-4o-mini`
3. 点击【测试连接】，显示绿色"成功"即完成

> 💡 **小提示**：如果地址以 `/v1` 结尾，星叙会自动拼接 `/chat/completions`，不要手动补全。

**高级API设置：**

- **Temperature（创意度）**：0~1.5，越高越天马行空，推荐 0.7~0.9
- **Max Tokens（输出长度）**：建议 4096~8192
- **流式输出**：开启后生成内容实时显示，关闭则等待完整结果后一次显示

**工具API（可选）：**

在API设置页可以单独配置"工具API"，用于 AI 评价、深度校对、角色设计等辅助工具，与主创作API分开，避免算力冲突。

**API预设：**

配置好一套API后，点击「保存API预设」可以给它起名保存。写不同类型故事时可以快速切换不同的API配置，保存后在预设列表中立即可见。

---

### 1.2 认识你的个人空间

**设置个人卡片：**

- **头像**：点击头像可上传图片，支持圆形裁剪界面——触摸拖动调整位置，双指/按钮缩放，确认后头像不再变形
- **昵称**：点击名字旁边的铅笔图标修改昵称
- **简介**：点击简介区域直接编辑

**查看写作统计：**

个人卡片下方有一行统计数字（总字数 / 作品数 / 收藏数 / 笔记数），**点击这一行**即可打开写作统计弹窗，查看作品字数占比饼图和最近7天每日创作字数折线图。点击上方名字或头像区域不会触发统计弹窗，方便单独修改资料。

**认识星笺（左上角小精灵）：**

在设置页个人卡片的左上角，有一个悬浮的紫色小头像，那就是**星笺**。轻轻点击即可打开星笺聊天页面。

星笺是整个网页的官方使用向导，随时可以问她：
- "怎么配置API？"
- "续写总是跑偏怎么办？"
- "这个按钮是干嘛的？"

## 第二章：写你的第一篇文章

1. 在「作坊」页面输入你的第一个灵感
2. 选择类型：**长篇连载** 或 **短篇一发**
3. 添加标签（甜文/悬疑/古风…）并调整创意度滑块
4. 点击【生成】，见证AI的第一次创作
5. 满意的作品点击【收藏】送入书架

> 💡 输入公式：**人物 + 冲突 + 背景 + 氛围**，效果比随手一句话好很多。

## 第二部分：核心功能

## 第三章：作坊

作坊是AI写作的主入口，一切创作从这里开始。

### 3.1 输入的艺术

**好Prompt的公式：**

```
角色 + 场景 + 冲突 + 风格 + 约束
```

**示例：**  
"林晚是一个社恐女生，在书店被高中同学认出，她不想承认。结局要温暖，可以带点幽默，不要写霸凌情节。"

---

### 3.2 智能标签系统

- 点击或自定义标签（甜文、悬疑、古风、科幻…）
- 标签会影响AI的创作基调和氛围
- 右上角菜单可管理标签

---

### 3.3 生成后的文章处理

- **收藏**：红心图标，点击后立即变化，文章进入书架
- **重新生成**：不满意时修改Prompt后重试
- **更名**：长按文章卡片 → 可自定义标题，告别AI乱取的名字
- **置顶**：长按 → 置顶，始终显示在列表最前面，方便管理进行中的项目
- **查看生成日志**：开发者模式下可查看完整提示词（用于排查问题）

---

### 3.4 快捷短语

长按输入框或点击右上角菜单，可调出**快捷短语**功能，快速插入常用提示词片段，省去反复输入的麻烦。

## 第四章：收藏

### 4.1 长篇书架管理

- **封面设置**：8种内置风格（深紫星辰、暖橙晚霞、翡翠森林、冰蓝深海、玫瑰粉黛、墨色素雅、金色宫廷、浅色简约）+ 自定义上传图片
- **状态管理**：连载中 / 已完结 / 暂停更新
- **星级评分**：为自己的作品打分，方便整理
- **排序与筛选**：按更新时间、字数、评分快速定位

### 4.2 短篇抽屉管理

- 快速浏览与搜索
- 复制、删除单篇
- 批量模式：多选后一键批量删除或导出TXT

---

### 4.3 收藏夹标签分类 ✨ v1.4.4 新功能

**入口：** 右上角「更多」→「⭐ 收藏夹」

搜索框下方新增了分类筛选栏，可以为收藏的段落打上标签，方便分类管理。

**新建分类：**

点击筛选栏末尾的虚线「＋新建」胶囊，在弹出的输入框中输入分类名称（不会弹出浏览器弹窗，使用App内置输入框）。

**为段落打标签：**

点击每条收藏卡片上的 🏷 图标，底部弹出分类选择器，勾选想要归类的分类，点「完成」关闭。一条段落可以属于多个分类。

**按分类筛选：**

点击筛选栏中任意分类胶囊，列表只显示该分类下的内容；点「全部」恢复显示所有收藏。

**管理分类：**

长按筛选栏中的分类胶囊，弹出「重命名/删除分类」选项，全程使用App内置弹窗。

**书名实时同步：**

原书修改标题后，收藏夹中展示的来源书名会自动更新，不会停留在旧书名。

**跳转原文保护：**

点击「跳转原文」时，系统会先校验原段落内容是否仍然存在。若内容已被修改，会提示"原文内容已改变，无法跳转"，不会跳到错误位置。

### 5.2 编辑器功能全解析

**基础功能：**

- 保存（Ctrl+S）、排版、字数统计与Token估算
- 写作排版设置：自定义字号、行高、字距
- 搜索定位（Ctrl+F）
- 撤回与重做（最多30步）
- 沉浸写作：全屏无干扰模式

> 💡 **Token实时估算（v1.4.4）**：编辑器底部会实时显示当前文本+系统提示词的预估Token数。上下文编辑器底部有「📊 Token估算」区域，点击「刷新」可查看各部分消耗和总使用率。使用率超过85%会显示黄色警告，提醒你精简提示词或使用精华压缩器。

**章节管理（小说模式）：**

添加、删除、重命名章节，章节拖拽排序。

**版本管理：**

保存历史版本，随时回溯，对比差异；不满意时保存版本后大胆重写再对比。

### 5.3 氛围写作

在编辑器「更多工具」中点击「☁️ 氛围写作」开启。

**五种氛围：**

| 氛围 | 音效特色 |
|------|------|
| 🌧 雨天 | 淅沥雨声 + 偶尔翻书声 |
| ☕ 咖啡店 | 暖意背景音乐，慵懒轻松 |
| 🌲 森林 | 鸟鸣风声 + 萤火虫光点 |
| 🏠 雪天小屋 | 暴风雪 + 壁炉噼啪声 |
| 📚 图书馆 | 书写声与翻书声 |

选择写作时长（25/45/60/90分钟或自定义）后开始。进入后：
- 屏幕上方显示计时进度条
- 右下角有计时器浮标
- 左下角半透明取消按钮

**自定义（设置 → 其他设置 → 沉浸式写作）：**

- 特效强度：控制雨滴/雪花/萤火虫密度

## 第六章：设置

### 6.1 人物设定

- **创建与编辑角色**：姓名、性格标签、重要度（核心/普通/参考）、详细设定
- **人物关系网**：绘制关系图，支持方向标注和关系描述
- **可视化图谱**：所有角色排成圆形，彩色连线代表关系类型，点击节点查看信息卡片，可按关系筛选
- **NPC管理**：管理配角和路人角色
- **人物预设**：保存并在不同故事间复用，保存后立即显示在预设列表

**重要度分级的作用：**

- **核心**：AI将设定视为铁律，严格执行
- **普通**：保持前后一致即可
- **参考**：可灵活处理，AI有发挥空间

---

### 6.2 世界观设定

- **全局世界观**：适用于所有故事的背景
- **局部世界观**：绑定到特定人物预设，切换预设时自动切换
- **每章最少字数**：约束AI不能写太短
- **内置模板**：现代都市、古代宫廷、仙侠修真、西方奇幻、末日废土、民国乱世等一键套用

---

### 6.3 文风、规则与禁用词

**文风设置：**

- 定义写作风格（甜宠、悬疑、古风等）
- 支持绑定角色预设
- 内置模板：轻松日常、沉郁文学、甜宠言情、悬疑紧张、古风典雅、幽默讽刺、硬核写实等

**规则设置：**

- **全局规则**：所有故事始终生效（如"结局必须HE"）
- **局部规则**：绑定特定预设时生效（如"古风限定：禁止使用现代词汇"）

**禁用词设置：**

- 全局禁用词：屏蔽你最讨厌的词汇
- 局部禁用词：针对特定项目单独配置
- 建议少而精，内置常见网文通病模板可一键套用

---

### 6.4 自定义系统提示词

- **注入位置**：
  - **头部**：优先级更高，会覆盖默认指令
  - **尾部**：适合补充说明
- 支持保存提示词预设，保存后立即可见

### 6.7 万能输入框 ✨ 新功能

**入口：** 设置 → 工具与模板 → 万能输入框（简洁模式下隐藏，标准模式可见）

**适合什么情况用？**

当你脑子里有故事想法，但不知道哪些设定该填在人物、哪些填在世界观时，直接用万能输入框倾倒想法，AI帮你自动归类。

**操作流程：**

1. 在文本框里随便写任何内容，例如：
   - "林晚，女，22岁，社恐，喜欢读书，在古代书店邂逅了顾深……"

- **保存时同步绑定预设**：保存模板前如果已使用了世界观/文风/规则/禁用词预设，该预设名称会一并记录进模板；模板卡片上会显示"世界观:古风宫廷"等具体名称，而不是只显示"世界观✓"

- **分类改名/删除**：长按分类标签，可重命名或删除分类

## 第十章：今日一写

**入口：** 右上角「更多」→「✍ 今日一写」（也可在设置页面找到入口）

今日一写是专为坚持创作设计的轻量写作工具。每天只需写一段，积少成多，培养写作习惯。

### 10.1 核心功能

- **今日灵感**：每次打开从灵感库（内置100条）随机抽取一条写作提示，如"一封永远不会寄出的信"。点「换一个」可重新随机，**每天固定同一条**，不会变来变去
- **AI帮我开个头**：点击后AI根据当天灵感生成约100-150字的精彩开头，直接填入写作区，你接续即可

## 第十四章：常见问题

### 14.1 API相关问题

**测试连接总是失败？**

1. 检查API地址是否以 `https://` 开头，无多余空格
2. 注意：地址若以 `/v1` 结尾，不要再写 `/chat/completions`
3. 密钥是否完整、正确、未过期
4. 国内访问官方OpenAI通常需要代理

**常见错误码：**

| 错误码 | 含义 | 解决方法 |
|--------|------|----------|
| 401 | 认证失败 | 检查API密钥是否正确，有无多余空格 |
| 404 | 接口路径不存在 | 检查API地址，是否多写/v1 |
| 429 | 请求频率过高/额度不足 | 稍等片刻再试，检查账户余额 |
| 流式报错 | 网络不稳定或不支持流式 | 关闭「流式输出」开关后再试 |

**为什么提示"操作太频繁"？**

每次AI操作后有2秒冷却时间，防止重复点击造成费用激增。等待2秒后再试即可。

---

### 14.2 生成质量问题

**续写总是跑偏（OOC）？**

- 降低创意度到 0.6~0.7
- 在「上下文编辑器」中写清楚人物当前状态和本章大纲
- 使用「🧠 智能续写」代替快速续写
- 完善人物设定（重点写性格标签和详细描述）
- 设定太长时，用「精华压缩器」压缩

**生成的文章像流水账？**

- 输入中加入"转折""冲突""悬念"等关键词
- 使用标签引导氛围
- 生成后使用「AI评价」获取修改建议，再用「AI重写」优化

**字数总是达不到要求？**

- 在「设置 → 世界观设定」中调高"每章最少字数"
- 生成后使用「AI扩写」
- 如果续写字数不足，系统会自动尝试补写

---

### 14.3 性能与数据问题

**页面卡顿？**

- 开启"简洁模式"（设置 → 页面美化）
- 在「设置 → 数据管理 → 清理工具」中清理旧文章、旧日志
- 存储空间快满时，导出备份后清理

**不小心删除了重要作品？**

- 有备份：直接导入恢复
- 无备份：尝试「紧急数据恢复」（设置 → 数据管理）——越早执行成功率越高

**生成过程中断怎么办？**

星叙在生成时支持后台保活（需浏览器支持WakeLock API）。如果生成被意外中断，下次续写时会自动检测断点，询问是否从中断处恢复。断点每200字保存一次。不想用断点可选择"丢弃断点，重新生成"。

**如何让上下文设置自动保存？**

在「设置→其他设置→阅读与续写」中开启「保存所有上下文设置」开关（v1.4.4 新增）。开启后，关闭上下文设置面板时会自动保存前情提要、重要伏笔/悬念、特殊要求，不再需要每次点「保存并续写」才能留住。本章方向和大纲仍为章节级内容，此开关不影响这两项。
