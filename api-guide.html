§ 01 💡

基础概念，先搞清楚这几个词

🔑
API Key（密钥）
就像你账号的"通行证"，一串字母数字组合，是调用 AI 服务的唯一凭证。每个平台格式不同，OpenAI 以 sk- 开头，Gemini 以 AIza 开头。绝对不能泄露给他人！
sk-xxxxxxxxxxxxxxxx
🌐
API 地址（Base URL）
告诉程序"去哪个服务器"请求 AI。官方地址固定，中转站会有自己的地址。注意末尾是否需要 /v1，不同平台要求不同，填错会导致 404。
https://api.openai.com/v1
📦
模型名称（Model）
同一家服务商有多个 AI 模型，必须填写准确的模型 ID 才能调用到正确的模型。模型名区分大小写，一个字母写错就会报 404。
gpt-4o / claude-sonnet-4-20250514
💰
Token（令牌）
AI 的计费基本单位。英文约 1 个单词 = 1~2 token；中文约每 1.5 个字 = 1 token。每次对话都消耗 token，输入和输出都算，账户余额不足会报 429 或 402 错误。
约 750 中文字 ≈ 1000 token
🔁
中转站 / 代理站
国内访问不了官方 API？中转站架设在海外服务器，代你转发请求。接口格式和官方完全一致，只需把 Base URL 改成中转站地址，Key 也换成中转站提供的。
relay / proxy / mirror
🎁
公益免费站
由热心网友自费搭建，免费开放 API 调用。额度有限、速率较低，适合临时测试或轻度体验。不稳定，随时可能停服，切勿用于正式创作。
free / public / community
🔒 安全红线——必读 API Key 等同于你的账号密码，一旦泄露他人即可盗用你的额度：
① 不截图发给任何人  ② 不存入公开文档/语雀/飞书  ③ 不提交到 GitHub 代码仓库
④ 如果怀疑已泄露，立即登录官网控制台将该 Key 撤销（Revoke），然后重新创建一个
§ 02 🤖

OpenAI（ChatGPT）

🤖
OpenAI 官方 官方直连 需科学上网
全球最强 GPT 系列 · GPT-4o、o3、o4-mini 等旗舰模型
OpenAI 是目前综合能力最强的 AI 平台，但国内无法直接访问，需要科学上网。新账号有少量免费额度，用完后须绑定境外信用卡付费（Visa/MasterCard，需非中国大陆发卡行）。
  1. 1
    开启科学上网并切换到全局代理模式
    必须使用全局模式（不是 PAC/规则模式），否则部分请求仍走国内网络会被拒绝。建议使用美国或欧洲节点，部分亚洲节点也会被 OpenAI 封锁。
  2. 2
    注册 OpenAI 账号
    访问 platform.openai.com，点击 Sign Up。
    建议使用 Gmail 或 Outlook 邮箱注册，国内邮箱和手机号均无法完成注册。如提示所在地区不支持，尝试清除 Cookie 或更换节点后重试。
  3. 3
    进入 API Keys 页面创建密钥
    登录后点击左侧菜单 API Keys，然后点击右上角 Create new secret key，在弹窗中为这个 Key 起一个名字（便于日后管理），点击创建。
    Key 只显示一次，请立即复制并妥善保存，关闭弹窗后无法再查看
  4. 4
    充值或确认账户余额
    前往 Billing → Add payment method 绑定境外信用卡，然后充值（最低 $5）。免费额度耗尽前可以先测试连通性。
  5. 5
    填写配置信息
    将以下信息填入星叙的 API 配置：
    Base URL https://api.openai.com/v1
常用模型 gpt-4o # 旗舰，速度与质量平衡（推荐）
gpt-4o-mini # 经济版，价格约为 4o 的 1/10
o3-mini # 深度推理，解复杂逻辑问题
o4-mini # 最新一代推理模型
✓ 综合能力最强 ✗ 需科学上网 ✗ 需境外信用卡 $ 按量付费
§ 03 🧠

Anthropic Claude

🧠
Anthropic 官方 官方直连 需科学上网 写作首选
Claude Sonnet / Haiku 系列 · 长文写作、文风模仿、角色扮演能力出众
Claude 在长文写作、文风模仿、创意角色扮演方面表现最为出众，是创作类应用的热门选择。对 IP 地址要求较严格，部分亚洲节点会被拒绝，建议使用美国或欧洲节点。
  1. 1
    开启科学上网，切换到美国/欧洲节点
    Claude 对 IP 要求比 OpenAI 更严格，香港、台湾、部分东南亚节点可能被拒绝，建议优先选择美国节点，且使用全局代理模式。
  2. 2
    注册 Anthropic 账号
    访问 console.anthropic.com，用邮箱注册。
    手机号验证支持国内号码（+86），但注册时的 IP 必须是允许地区，否则会直接报错。如遇注册页面空白，请尝试清除缓存或换节点。
  3. 3
    创建 API Key
    登录后点击左侧 API Keys，点击 Create Key，为 Key 命名后点击创建。
    同样只显示一次，请立即复制保存
  4. 4
    充值账户余额
    进入 Billing 页面，绑定境外信用卡并充值（最低 $5）。新账号会赠送少量免费额度用于测试。
  5. 5
    填写配置信息
    注意 Claude 的 Base URL 末尾没有 /v1：
    Base URL https://api.anthropic.com
常用模型 claude-sonnet-4-20250514 # 最新旗舰写作模型（推荐）
claude-haiku-4-5-20251001 # 轻量快速版，经济实惠
接口格式注意 Claude API 格式与 OpenAI 略有不同（如无 model 前缀等），如果星叙是基于 OpenAI 格式开发的，建议选择 Claude 专属接口模式，或通过中转站使用 OpenAI 兼容的 Claude 接口。
✓ 写作创作最强 ✓ 长文理解好 ✗ 需科学上网 ✗ IP 要求严格 $ 按量付费

Google Gemini

💎
Google AI Studio（官方） 官方 有免费额度 性价比高
Gemini 2.5 Pro / Flash · Flash 完全免费，性价比极高
Google AI Studio 每天有免费请求额度，Gemini 2.0 Flash 完全免费。性价比极高，2.5 Pro 推理能力已达 GPT-4 级别。注意需使用非中国区的 Google 账号（个人账号即可，无需 Workspace）。
  1. 1
    准备非中国区 Google 账号
    注册时选择非中国地区（如美国/日本）的 Google 账号，或将已有账号的地区在 Google 账号设置中修改。中国区账号无法访问 AI Studio。
  2. 2
    开启科学上网，访问 AI Studio
    访问 aistudio.google.com，用 Google 账号登录。如果看到地区不支持的提示，尝试更换节点或清除 Cookie。
  3. 3
    获取 API Key
    点击页面左上角或顶部的 Get API key，选择 Create API key，选择或新建一个 Google Cloud 项目，Key 即刻生成（格式为 AIza 开头）。
    Gemini 的 Key 可以随时在控制台重新查看，无需担心丢失
  4. 4
    填写配置信息（使用 OpenAI 兼容接口）
    Gemini 提供了专门的 OpenAI 兼容地址，注意末尾有 /openai/：
    Base URL https://generativelanguage.googleapis.com/v1beta/openai/
常用模型 gemini-2.0-flash # 完全免费，速度快（推荐入门）
gemini-2.5-flash-preview-05-20 # 免费额度内可用，更智能
gemini-2.5-pro-preview-06-05 # 旗舰推理（需付费）
✓ Flash 完全免费 ✓ OpenAI 兼容 ✓ Key 可重复查看 ✗ 需科学上网 $ 免费 + 付费双档

DeepSeek（深度求索）

🔍
DeepSeek 官方 国内直连 超高性价比 新号送额度
国产最强模型 · 接近 GPT-4 水平 · 价格约为 GPT-4o 的 1/30 · 无需科学上网
DeepSeek V3 和 R1 是目前国内最受欢迎的 AI 模型，综合能力媲美国际顶尖水准，但价格极低。国内用户首选，10 元可以用很久。接口完全兼容 OpenAI 格式，迁移成本极低。
  1. 1
    直接访问 DeepSeek 平台
    无需科学上网，直接访问 platform.deepseek.com，点击右上角注册。
  2. 2
    手机号注册并完成实名认证
    输入国内手机号获取验证码，设置密码完成注册。部分功能需要完成实名认证（上传身份证），认证后可获得更高的免费额度。
  3. 3
    充值或使用新用户赠送额度
    新用户注册后系统会赠送一定免费 token（金额不定，视活动而定）。如需正式使用，前往 充值 页面，支持支付宝/微信支付，最低充值 10 元。
    10 元约可生成数十万 token，普通用户用几个月都没问题
  4. 4
    创建 API Key
    点击左侧菜单 API Keys，点击 创建 API Key，填写名称（便于区分用途），点击确认。
    同样只显示一次，请立即复制保存
  5. 5
    填写配置信息
    Base URL https://api.deepseek.com/v1
常用模型 deepseek-chat # DeepSeek V3（综合能力强，日常推荐）
deepseek-reasoner # DeepSeek R1（深度推理，逻辑解题强）
强烈推荐 ⭐⭐⭐⭐⭐ 无需科学上网、价格极低、接口和 OpenAI 完全兼容——对国内用户而言，DeepSeek 是综合体验最优的起点。如果只想快速接一个 API 开始用，优先选这个。
✓ 无需科学上网 ✓ 价格极低 ✓ OpenAI 兼容 ✓ 人民币充值 $ ≈ 0.001 元/千 token

文心一言（百度）

百度智能云千帆平台 国内直连 部分免费
ERNIE 系列 · 中文语境理解好 · 适合中文内容创作与合规场景
百度文心 ERNIE 系列对中文语境理解较深，适合中文内容生成任务。接入方式比其他平台略复杂，但千帆平台已提供 OpenAI 兼容接口，可简化操作。
  1. 1
    访问千帆平台并登录

    开通模型服务并获取 API Key
    进入 模型广场 找到需要的模型（如 ERNIE-4.5）点击开通。然后前往 应用接入 → 创建应用 → 获取 API Key（推荐直接使用 v2 OpenAI 兼容 Key，无需再手动换 token）。
  4. 4
    填写配置信息（使用 OpenAI 兼容接口）
    Base URL https://qianfan.baidubce.com/v2

§ 08 🔁

中转站 / 代理站

什么是中转站？ 中转站架设在海外服务器，代替你向官方 API 发送请求。你无需科学上网，只需把 Base URL 改成中转站地址，把 Key 换成中转站提供的。接口格式和官方完全一致，可以同时接入 GPT、Claude、Gemini 多个模型。
🔁
中转站接入方式 第三方代理
收费型 · 稳定可靠 · 支持 GPT / Claude / Gemini 等多模型
中转站一般按 token 售卖，价格通常比官方略贵（包含代理服务成本），但省去科学上网的配置。寻找推荐可以去各大 AI 交流群、V2EX、少数派、NodeSeek 等社区搜索口碑良好的站点。
  1. 1
    找到可靠的中转站并注册账号
    在 AI 社区通过口碑评价筛选，优先选择运营超过 6 个月、有活跃用户反馈的站点。先用小额充值（10~20 元）测试稳定性，再决定是否继续使用。
    不要相信"永久有效额度"，大额预付风险很高
  2. 2
    在个人中心获取 API Key
    登录中转站后台，在"令牌"或"API Keys"页面创建一个 Key。注意这个 Key 是中转站发的，不是官方的，只能配合该中转站的地址使用。
  3. 3
    填写中转站 Base URL
    每个中转站的地址不同，通常格式为：
    https://api.某中转站.com/v1 # 格式示例
    具体地址在中转站的"接入文档"或"使用说明"中查找。
  4. 4
    填写模型名称并测试
    模型名称通常与官方一致（如 gpt-4o、claude-sonnet-4-20250514），部分中转站会提供专属模型列表，以站内文档为准。发送"你好"测试连通性。
使用中转站须知 ① 你的对话内容会经过第三方服务器，请勿发送个人隐私、商业机密
② 避免选择要求大额预付的站点
③ 中转站随时可能停服，建议保留官方备用方案
④ 挑选时优先参考 AI 社区中有使用记录的真实口碑
✓ 无需科学上网 ✓ 支持多种顶级模型 ✗ 存在跑路风险 ✗ 隐私风险 $ 略贵于官方直连
§ 09 🎁

公益免费站

关于公益站 由热心网友自费搭建，免费开放 API 调用。通常限速、限每日额度，适合零成本轻度体验或功能测试。请珍惜公共资源，不要滥发请求！
🎁
公益免费站特点 完全免费
无需付费 · 注册即用 · 每日额度有限
公益站通常基于 One API / New API 等开源项目搭建，接口与 OpenAI 格式完全兼容，支持 GPT-4、Claude、Gemini 等多种模型，每日额度自动刷新。
  1. 1
    寻找公益站地址
    在相关 Telegram 群组、QQ 群、或 GitHub 上搜索"OpenAI 公益 API"、"免费 API 站"等关键词。由于公益站随时可能停服，建议加入相关社群以获取最新可用地址。
  2. 2
    注册账号（通常只需邮箱）
    大多数公益站只需邮箱即可注册，部分需要邀请码。注册后到"令牌"或"API Keys"页面创建一个 Key。
  3. 3
    填入 Base URL 和 Key 测试
    将公益站提供的 Base URL 和 Key 填入星叙，发送"你好"测试是否正常。如果当日额度耗尽会返回 429 错误，次日刷新后恢复。
    公益站平均寿命较短，遇到停服时及时寻找替代
重要提醒 公益站对话内容经过第三方服务器，切勿输入个人隐私、密码、商业机密等敏感信息。稳定性无法保障，不建议用于重要创作场景。
✓ 完全免费 ✓ 无需科学上网 ✗ 不稳定 ✗ 额度有限 ✗ 随时停服 ✗ 隐私风险高

将 API 配置到「星叙」

获取到 API Key 和 Base URL 之后，按以下步骤填入星叙即可开始使用。全程不超过 2 分钟。

各平台 Base URL 速查（点击可复制）
DeepSeekhttps://api.deepseek.com/v1
通义千问https://dashscope.aliyuncs.com/compatible-mode/v1
Geminihttps://generativelanguage.googleapis.com/v1beta/openai/
OpenAIhttps://api.openai.com/v1
Claudehttps://api.anthropic.com

打开星叙 → 设置 → API 配置

点击底部导航栏的「设置」图标，找到「API 配置」或「AI 服务」入口，进入配置页面。

填写 API 地址（Base URL）

将对应平台的 Base URL 粘贴到「API 地址」输入框。注意不要在末尾多加斜杠，也不要缺少 /v1（Claude 除外）。如有疑问参考上方速查表。

填写 API Key

将密钥完整粘贴到「API Key」输入框。注意不要有前后多余的空格——这是最常见的错误之一。粘贴后建议肉眼确认一下开头和结尾字符是否正确。

填写模型名称（Model）

在「模型」输入框填入对应的模型名。建议先填轻量快速的模型（如 deepseek-chat 或 gpt-4o-mini）进行连通性测试，确认无误后再换旗舰模型。

  • 模型名区分大小写，完全匹配官方文档中的写法
  • 不确定模型名时，去对应平台的「模型广场」或官方文档查询

保存并发送测试消息

点击保存后，在对话框输入「你好」发送。如果 AI 正常回复，说明配置成功！

  • 如果报错，优先检查：Key 是否复制完整（无空格）、URL 格式是否正确
  • 出现 401 错误 → Key 有问题；404 错误 → URL 或模型名有误；429 错误 → 额度不足或频率过高
  • 遇到连接超时（国际平台）→ 检查科学上网是否开启
§ 12 🛠

常见错误 & 排查方法

401 Unauthorized 认证失败 · API Key 无效
这是最常见的错误，通常是密钥问题导致的。按以下顺序逐一排查：
  • Key 是否完整复制？确认开头结尾字符无误，无多余空格或换行
  • Key 是否已被撤销或过期？登录平台控制台查看 Key 状态
  • Key 和 Base URL 是否匹配？DeepSeek 的 Key 不能配 OpenAI 的地址使用
  • 账号是否欠费？部分平台账户余额不足也会返回 401
429 Too Many Requests 请求频率超限 / 额度不足
触发了平台的速率限制，或账户余额已耗尽。
  • 检查账户余额是否充足，及时充值
  • 短时间内请求太频繁，等待 30~60 秒后重试
  • 公益站每日额度耗尽，等次日刷新
  • 检查是否有多个设备/软件同时使用同一个 Key
404 Not Found 地址或模型不存在
路径配置有误，或填写的模型名称不正确。
  • Base URL 末尾是否多了/少了斜杠？（如 …/v1 vs …/v1/）
  • Claude 的 Base URL 末尾不带 /v1，其他平台通常需要
  • 模型名是否拼写完全准确？大小写、连字符一个都不能错
  • 去平台官方文档"模型列表"核对正确的 Model ID
连接超时 / 无响应 网络无法到达服务器
通常是访问国际 API 时没有开启科学上网，或代理未正确工作。
  • 确认科学上网软件已开启，且处于全局模式（不是 PAC 规则模式）
  • 尝试更换节点（美国或欧洲），部分亚洲节点被 OpenAI/Claude 封锁
  • 国内平台（DeepSeek/通义/文心）不需要科学上网，如果超时检查本地网络
  • 改用中转站，彻底避免代理问题
模型名称问题 不知道该填什么 Model
模型名必须和平台文档完全一致，以下是几个快速查找的方法：
  • 去各平台的"模型广场"或"API文档"页面，找"模型列表"，复制 Model ID
  • 注意区分中文显示名（如"文心大模型4.5"）和 API Model ID（如 ernie-4.5-turbo-128k）
  • 先用一个肯定存在的轻量模型测试连通性，再换目标模型
配置成功但回复异常 AI 回复乱码 / 截断 / 拒绝
连通正常但响应质量有问题，通常是模型设置或提示词问题。
  • 乱码 → 检查字符编码设置，或该模型对中文支持较弱，换模型试试
  • 回复截断 → max_tokens 设置过小，在高级设置中调大
  • 拒绝回复 → 触发了模型的内容过滤，调整提示词
