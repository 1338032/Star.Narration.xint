使用流程

四步
开始你的第一篇故事

01
配置 API
填写 API 地址、密钥、模型名称，点击测试连接，绿灯亮起即可使用。支持 OpenAI / Claude / Deepseek 等主流接口。
02
写下灵感
在作坊输入故事设想，选择标签，调整创意度，长篇短篇任选。卡文时打开故事转盘，随机获取情节触发点。
03
收藏 & 续写
对结果满意就收藏进书架，随时续写下一章，上下文设置让情节不跑偏，伏笔追踪器记录每条线索。
04
精修 & 导出
润色、重写、做章节笔记、追踪伏笔，完成后导出 TXT，或打包资源包分享给创作伙伴。

数据永不被绑架

TXT 随时导出 选中任意章节或整部作品，一键导出纯文本，投稿、排版、发布任你用，不依赖平台格式。
传输全程本地 所有导入导出操作均在本地完成，文件不经过任何服务器，你的故事只属于你。

接入指南

三步接入 API
立刻解锁 AI 能力

星叙不内置模型，由你自选 API 服务商，完全控制费用与模型版本。

OpenAI / ChatGPT
Claude / Anthropic
Deepseek
硅基流动
任意 OpenAI 兼容接口
1
打开设置页 → API 配置
点击底部导航栏「设置」图标，找到「API配置」一栏，点击展开配置面板。
2
填写 API 地址、密钥、模型名称
粘贴你的 API Base URL（如 https://api.openai.com/v1），填入 API Key，选择或输入模型名称（如 gpt-4o-mini）。
3
点击「测试连接」，绿灯亮起即可
星叙会发送一条测试请求验证连通性。测试通过后，所有 AI 功能立刻解锁，密钥仅保存在你的本地设备。
